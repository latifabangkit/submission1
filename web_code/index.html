My Story about Joining Bangkit

Halo!! Saya merupakan siswa Cloud Computing di program Bangkit Academy 2023 yang diseleranggakan oleh Google, Goto, dan Traveloka. Saya merupakan mahasiswa jurusan Matematika di Universitas Indonesia. Saya memilih Bangkit dikarenakan Bangkit sudah terpercaya dan memiliki lulusan yang sukses. Bangkit juga menyediakan berbagai kelas dan pelatihan baik secara akademis maupun non-akademis. Selain itu, Bangkit memberikan kesempatan bagi setiap siswa untuk mengikuti ujian sertifikasi dan tentunya diberikan pembelajaran dan bekal agar setiap siswa dapat lulus pada ujian sertifikasi. Melalui program Bangkit ini, saya telah mempelajari banyak hal yang dapat menunjang skill dan persiapan karir saya nantinya.
